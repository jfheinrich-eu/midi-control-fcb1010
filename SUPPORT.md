# Support

## Getting Help

For setup or usage questions:
- Open a GitHub Discussion (preferred for Q&A)
- Or open a GitHub issue if you are unsure whether it is a bug

## Bug Reports

For confirmed bugs, use the Bug Report issue template and include:
- Cubase version
- Controller firmware/setup details
- MIDI mapping details
- Reproduction steps

## Feature Requests

Use the Feature Request template and describe:
- Your workflow problem
- Desired behavior
- Why existing behavior is insufficient
