# Security Policy

## Supported Versions

This project currently supports the latest `main` branch state.

## Reporting a Vulnerability

Please do not open public GitHub issues for security-sensitive reports.

Report vulnerabilities by email to:
- contact@jfheinrich.eu

Please include:
- A clear description of the issue
- Affected file(s) and behavior
- Steps to reproduce
- Potential impact

We will acknowledge receipt and evaluate the report as quickly as possible.
